  1. Qus: What are three components of the web?

    Ans: HTML, CSS & Javascript

  2. Qus: Define Control and give one example

    Ans: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. An example is GitHub

  3. Qus: What tag is used to create the title of a webpage? Give an example.

    Ans: <title> An example is <title>QA</title>

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: <p> An example is <p>paragraph</p>

  5. Qus: Identify the odd one out from these tags: <span>, <p>, <a>, <i> and state the reason.

    Ans: <span>because it is an inline container.

  6. Qus: What tags are required to create a navigation list html?

    Ans: <nav>, <ul>, <ol>, <li>

  7. Qus: What tags are used to create images? Name it's attributes.

    Ans: <img> its attributes are "alt, height, width and src"

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td>

    Ans: <b>because it is used to make things bold and all the other tags relate to tables

  9. Qus: Give and example of colour picker input

    Ans: Hex code #FEFF33 or RGB 254, 255, 51

  10. Qus: What is the use of the name and value attributes? Give two examples

    Ans: The name attribute specifies a name for an HTML element. Examples <button> <form> The value attribute specifies the intial value of the element. Examples <button> <li>

  12. Qus: How would you link an external style sheet? Give an example

    Ans: You would add a link to it in your header. The following is an example: <link rel="stylesheet" type="text/css" href="css/style.css">

  13. Qus: Create CSS block to center an element with both horizontally and vertically that has position:absolute and has width:200px and height200px

    Ans: .loader { position: absolute; left: 50%; top: 50%; width: 200px ; height: 200px ; }

  14. Qus: Create CSS keyframes block called "spin" that rotates an element

    Ans: @keyframes spin { 100% {transform:rotate (360);} }

  15. Qus: Create CSS animation block to rotate an element that has the following properties Class "rotate me" & It rotates 5 times at intervals of o.5s with "linear" transition using animations keyframes "spin"

    Ans: .rotate-me { animation spin .5s linear 5} @keyframe spin { 100% {transform: rotate(360 deg);}

  16. Qus: How would you include create an alert with string "I love JavaScript" using Javascript?

    Ans: <script type="text/javascript"> alert ("I love JavaScript")</script>

  17. Qus: How would you fade out of an element with id "fade-me" after a delay of 300ms using JQuery?

    Ans: $("#fade-me").delay(300).fadeOut();

  18. Qus: How would you select all elements with class "my-class" using Javascript and Jquery

    Ans: $(".my-class")

  19. Qus: Give an example of input field that takes phone number with format "123-456-7890"?

    Ans: <input type="tel id="phone name="phone" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" >
